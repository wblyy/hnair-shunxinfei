≠
顺心卡 F/Q 舱位开放

顺心卡只认指定 F/Q booking class。F/Q 不开，就算普通票在卖也兑不了。

用户话术：“这班不是没票，而是顺心卡要用的那个舱位没放出来。”

关键修正：顺心卡看的是“可兑舱位池”

F/Q 是 booking class / fare bucket，不是简单按“头等/经济”的直觉理解。航班 APP 有票、经济舱有座、甚至低价票在售，都不等于顺心卡可以兑换。

正确评估应按 航司 → 航班 → 日期/提前期 → 指定 F/Q 舱位 → 官方/黑屏/抓取结果 分层判断。
