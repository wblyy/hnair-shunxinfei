先验航班，再验直飞，最后才看 F/Q

这版不再一上来就判“F/Q 未开”。先确认海航 APP/官方渠道查得到、是否直飞单航班、是否属于适用航司，再判断普通票和 F/Q 舱位。

1APP 是否查得到
2是否直飞单航班
3普通票是否在售
4F/Q 是否开放

主页最关键判断：不要一上来就看 F/Q

只有“APP 可见 + 直飞单航班 + 普通票在售 + F/Q 未开放”同时成立，才叫“能买但不能兑”。

如果 APP 查不到，这不是“没仓”，而是航班/行程不在可验范围；如果是中转组合，就不能按直飞单航班验 F/Q。

1APP 可见海航 APP / 官网能否查到该航班或行程。
2直飞确认必须是直飞单航班；中转不能拿来验单航班 F/Q。
3适用航司承运航司是否属于金鹏顺心卡适用范围。
4普通票先看普通票池是否在售，避免把查无航班误判成没仓。
5F/Q 舱HU/CN/Y8 看 F；其余适用航司看 Q。
6最终判断最后才输出“能买但不能兑”或“可兑机会”。

三个不同结论，不能混在一起

APP 未见航班/待核验：不是没仓，是这个航班/行程可能不在海航 APP 可查或顺心卡可用渠道内。

中转组合：不是直飞单航班，不能直接按 F/Q 单航班验仓。

普通票可能在售，但 F/Q 未见开放：这才是典型“能买票但顺心卡兑不了”。

舱位映射规则

F 舱：海南航空 HU / 大新华 CN / 金鹏航空 Y8

Q 舱：首都 JD / 天津 GS / 祥鹏 8L / 西部 PN / 北部湾 GX / 乌航 UQ / 福航 FU / 长安 9H

F/Q 是 booking class / fare bucket，不等于普通用户看到的“还有座”。
